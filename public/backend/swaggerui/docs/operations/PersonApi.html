peopleGet

List of the people that works in the Association

URL

http://localhost:8080/v2/people

HTTP Method

GET

Response Type

List

Parameters

  • Query: offset Integer

    Pagination offset

  • Query: limit Integer

    Maximum number of item per page. Default is 20.
